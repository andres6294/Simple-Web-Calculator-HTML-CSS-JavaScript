# Simple-Web-Calculator-HTML-CSS-JavaScript
This project is a basic web calculator created as a personal learning exercise. The focus was to practice DOM manipulation, event handling, and styling with CSS, while keeping the design clean and functional.

Features
- Addition, subtraction, multiplication, and division
- Disabled output field to prevent manual edits
- Styled buttons and hover effects
- Centered calculator layout inside a styled container
- Light gradient background and subtle shadow for depth 
- Fully implemented using pure JavaScript (no external libraries)
